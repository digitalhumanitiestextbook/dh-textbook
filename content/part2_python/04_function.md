## 함수

지금까지 우리는 변수, 데이터 타입, 연산자, 제어문 등을 배우며 파이썬의 기본적인 작동 방식을 익혔습니다. 하지만 코드가 길어지거나 비슷한 작업을 여러 번 반복해야 할 때, 코드를 효율적으로 관리하고 재사용하는 방법이 필요합니다. 이때 사용하는 것이 바로 **함수(Function)**입니다.

함수는 특정 작업을 수행하는 **코드 묶음**에 이름을 붙여 놓은 것입니다. 마치 우리가 연구 과정에서 특정 분석 단계를 정형화하여 "OO 분석법"이라고 부르거나, 글쓰기에서 자주 사용하는 문장 패턴을 만들어 두는 것과 비슷합니다. 

한번 잘 만들어 둔 함수는 필요할 때마다 함수 이름으로 불러와 해당 작업을 쉽게 다시 수행할 수 있습니다. 함수를 사용하면 코드가 간결해지고, 수정하기 쉬워지며, 전체 프로그램의 구조를 이해하기 용이해집니다.

이 장에서는 파이썬에서 함수를 직접 만들고(정의하고) 사용하는(호출하는) 방법, 그리고 파이썬에 이미 내장되어 있는 유용한 함수들을 활용하는 방법을 '프렌즈' 관련 예시를 통해 배워 보도록 하겠습니다.

## 나만의 명령어 만들기: 함수 정의와 호출

**함수 정의(Definition)**는 특정 작업을 수행하는 코드 블록, 즉 '함수'에 이름을 붙여 주는 과정입니다. 파이썬에서는 `def` 키워드를 사용하며, 다음과 같이 정의할 수 있습니다.

```
def 함수이름(매개변수1, 매개변수2, ...):
    """독스트링(Docstring): 이 함수가 무엇을 하는지 설명하는 글 (선택 사항이지만 권장)"""
        # 함수가 수행할 코드 블록 (들여쓰기 필수)`
        # 필요하다면 return 키워드로 결과값 반환`
```

이렇게 정의한 함수는 언제든 호출해 정의된 기능을 활용할 수 있으며, 이때 **호출(Call)**은 정의된 함수의 이름을 불러 실제로 그 함수 안의 코드를 실행시키는 과정을 말합니다. 함수에는 여러 개의 인자(argument)가 들어갈 수 있으며, `함수이름(인자1, 인자2, ...)` 형태로 사용합니다. 인자에 대해서는 뒤에서 다시 살펴보도록 하겠습니다.


### 캐릭터 소개 함수 만들기 (정의 및 호출)

가장 간단한 형태로, 특정 캐릭터에 대한 고정된 소개 문구를 출력하는 함수를 실행시켜 봅시다.

`introduce_phoebe`라는 이름의 함수를 만들어서, `"피비는 독특하고 엉뚱한 매력의 소유자입니다."`를 출력하게 하고, 그 함수를 호출하는 코드 만들어 봅시다. 

다음은 `introduce_phoebe` 함수를 정의한 후에, 그 함수를 바로 실행(호출)시키는 코드입니다.

```
# 피비를 소개하는 함수 정의
def introduce_phoebe():
    """피비 캐릭터에 대한 간단한 소개를 출력하는 함수"""
    print("피비는 독특하고 엉뚱한 매력의 소유자입니다.")

    # 정의된 함수 호출 (실행)
    print("=== 피비 소개 ===")
    introduce_phoebe()
```

**실행 및 설명:**

* 결과 확인
    * `=== 피비 소개 ===`, `피비는 독특하고 엉뚱한 매력의 소유자입니다.`
    * `def introduce_phoebe():` 부분이 함수를 정의하는 코드입니다. 괄호 `()` 안이 비어있는 것은 이 함수가 외부로부터 정보를 받지 않는다는 의미입니다.
    * 들여 쓰기 된 `print(...)` 부분이 이 함수가 실제로 수행할 작업입니다.
    * `introduce_phoebe()` 부분이 정의된 함수를 호출(실행)하는 코드입니다. 함수 이름 뒤에 괄호 `()`를 붙여 호출합니다.


## 함수와 정보 주고받기: 매개변수와 Return 값

함수는 외부로부터 정보를 받아서 처리하거나(매개변수), 처리한 결과를 외부로 돌려줄(Return) 수 있습니다. 이를 통해 훨씬 유연하고 재사용성 높은 함수를 만들 수 있습니다.

**매개변수(Parameter)**는 함수를 정의할 때 괄호 `()` 안에 지정하는 변수로, 함수가 외부로부터 어떤 정보를 받을지 정의하는 '입력 틀'과 같습니다. 이는 마치 함수가 외부로부터 데이터를 몇 개 받기 위해 마련한 '자리표' 또는 '빈 그릇'과 같아서, 함수 본문 내에서 사용될 값을 임시로 저장하는 역할을 합니다.

한편 **인자(Argument)**는 함수를 호출할 때 매개변수에 실제로 전달하는 값입니다. 이는 '매개변수'라는 빈 그릇에 담기는 '내용물'이라고 생각할 수 있습니다. 함수를 호출할 때마다 이 인자의 값은 달라질 수 있습니다.

**반환(Return)** 값은 함수가 작업을 수행한 후, 그 결과를 함수를 호출한 곳에 되돌려주는 값입니다. `return` 키워드 뒤에 돌려줄 값을 명시합니다. `return` 문이 실행되면 반환과 함께 함수의 실행은 즉시 종료됩니다.

### 캐릭터 특징 설명 함수 (매개변수 활용)

캐릭터 이름과 특징을 입력받아 설명 문구를 출력하는 함수를 만들어 봅시다.

캐릭터 이름(`name`)과 그 캐릭터의 특징(`trait`) 두 가지 정보를 받아서, `"[이름]의 두드러진 특징은 [특징]입니다."` 형식의 문장을 출력하는 함수 `describe_character`를 정의하는 코드를 만들어 봅시다. 함수 정의 후, 이 함수를 `'로스'`와 `'공룡을 좋아함'` 정보를 넣어서 호출하고, `'모니카'`와 `'결벽증'` 정보를 넣어서 호출해 보겠습니다.

```
# 캐릭터 이름과 특징을 받아 설명하는 함수 정의
def describe_character(name, trait):
    """캐릭터 이름과 특징을 받아 설명 문구를 출력합니다."""
    print(f"{name}의 두드러진 특징은 {trait}입니다.")

    # 함수 호출 시 인자 전달
    print("=== 캐릭터 특징 설명 ===")
    describe_character("로스", "공룡을 좋아함") # name="로스", trait="공룡을 좋아함" 전달
    describe_character("모니카", "결벽증")     # name="모니카", trait="결벽증" 전달
```

**실행 및 설명**

* 결과 확인
    * 두 개의 설명 문장이 각각 출력됩니다.
    * `def describe_character(name, trait):` 에서 `name`과 `trait`이 바로 매개변수입니다. 이 함수는 두 개의 정보를 받아야 함을 의미합니다.
    * `describe_character("로스", "공룡을 좋아함")` 에서 `"로스"`와 `"공룡을 좋아함"`이 바로 인자입니다. 이 값들이 함수 내부의 `name`과 `trait` 변수에 각각 전달되어 `print`문에서 사용됩니다.

### 대사 길이 분류 함수 (Return 값 활용)

대사의 길이를 입력받아 '짧은 대사', '보통 길이 대사', '긴 대사'로 분류하고 그 결과를 돌려주는 함수를 만들어 봅시다.

구체적으로, 대사의 글자 수(`length`)를 입력받아서, 만약 길이가 10 미만이면 '짧은 대사', 10 이상 30 미만이면 '보통 길이 대사', 30 이상이면 '긴 대사'라는 문자열을 **결과로 반환(return)하는** 함수 `classify_dialogue_length`를 정의하는 코드를 작성해 보겠습니다. 함수 정의 후, 가상의 대사 길이 5, 20, 50을 각각 함수에 넣어서 호출하고, 돌려받은 결과를 출력하는 코드도 포함시켜 보겠습니다.

```
# 대사 길이를 받아 분류 결과를 반환하는 함수 정의
def classify_dialogue_length(length):
    """대사 길이에 따라 '짧은 대사', '보통 길이 대사', '긴 대사'로 분류하여 문자열을 반환합니다."""
    if length < 10:
        return "짧은 대사"
    elif length < 30: # 이미 위에서 10 미만은 걸러졌으므로, 여기는 10 이상 30 미만임
        return "보통 길이 대사"
    else: # 30 이상인 경우
        return "긴 대사"

    # 함수 호출 및 반환 값 출력
    print("=== 대사 길이 분류 ===")
    result1 = classify_dialogue_length(5)
    print(f"길이 5는? -> {result1}")

    result2 = classify_dialogue_length(20)
    print(f"길이 20은? -> {result2}")

    result3 = classify_dialogue_length(50)
    print(f"길이 50은? -> {result3}")
```

* **실행 및 설명**

* 결과 확인: 각 길이에 대한 분류 결과(`"짧은 대사"`, `"보통 길이 대사"`, `"긴 대사"`)가 출력됩니다.
    * 함수 내에서 `if/elif/else` 조건에 따라 다른 문자열을 반환(return)합니다. 
        * `return`은 함수의 실행을 종료하고 값을 호출한 곳으로 돌려줍니다.
    * `result1 = classify_dialogue_length(5)` 처럼 함수를 호출하면, 함수가 돌려준 값(여기서는 `"짧은 대사"`)이 `result1` 변수에 저장됩니다. 이 반환 값을 다른 계산에 사용하거나 출력할 수 있습니다.


## 파이썬이 미리 준비한 도구들: 내장 함수 활용

파이썬에는 우리가 직접 만들지 않아도 바로 사용할 수 있는 매우 유용한 **내장 함수(Built-in Function)**들이 많이 있습니다. 이미 우리는 `print()`와 `type()` 함수를 사용해 보았습니다. 몇 가지 더 알아봅시다.

* **`len(데이터)`**: 리스트, 딕셔너리, 문자열 등의 길이나 요소의 개수를 알려줍니다.
* **`str(값)`**: 주어진 값을 문자열(`str`) 타입으로 변환합니다.
* **`int(값)`**: 주어진 값을 정수(`int`) 타입으로 변환합니다. (숫자 형태의 문자열만 가능)
* **`float(값)`**: 주어진 값을 실수(`float`) 타입으로 변환합니다. (숫자 형태의 문자열이나 정수 가능)
* **`input(안내문구)`**: 사용자로부터 키보드 입력을 받습니다. (입력받은 값은 항상 문자열 타입)

그 외에도, 파이썬에는 도합 70여 개의 내장함수가 존재하며, 이는 파이썬 공식 문서에서 확인할 수 있습니다.

본격적인 실습에 앞서, 흔히 혼동하고는 하는 `input()` 함수에 대해 먼저 살펴보도록 하겠습니다.

### input()의 함정: 문자열만 받아요!

사용자에게 값을 직접 입력받는 `input()` 함수를 사용할 때 초보자들이 가장 많이 실수하는 부분이 있습니다. 바로 입력받은 '데이터 타입'입니다.

`input()` 함수는 사용자로부터 어떤 값을 입력받든, 즉 숫자 10을 입력하든, 문자열 ‘조이’를 입력하든, 그 결과는 **항상 문자열(str) 타입으로 반환**됩니다.

왜 이것이 문제가 될까요? `input()`으로 두 숫자를 입력받아 덧셈을 시도하는 경우를 생각해 봅시다. 그 경우, 파이썬은 숫자 계산이 아닌 문자열 결합을 수행하게 됩니다.

```
# 사용자에게 두 숫자를 입력받는다고 가정
# first_num = input("첫 번째 숫자: ")  # 사용자가 5 입력 -> "5" (문자열)
# second_num = input("두 번째 숫자: ")  # 사용자가 7 입력 -> "7" (문자열)

# 입력받은 두 숫자의 합 계산
# result = first_num + second_num

# print(result)  # 출력: "57"  (문자열 "5"와 "7"이 결합됨)
```

상기 코드를 실행시켜 보면, 반환되는 값은 `12`가 아닌, 문자열 `"57"`이 됩니다.

그렇다면 어떻게 이러한 문제를 해결할 수 있을까요? 사용자로부터 입력을 받아 덧셈, 뺄셈 등 숫자를 이용한 산술 연산을 수행하려면, `input()`으로 입력받는 즉시 해당 값을 `int()`나 `float()` 함수를 사용해 숫자 타입으로 강제 변환해야 합니다. 이를 우리는 **타입 변환(Type casting)**이라고 부릅니다.

```
# 입력과 동시에 정수(int) 타입으로 변환
first_num = int(input("첫 번째 숫자: "))
second_num = int(input("두 번째 숫자: "))

# 이제 정수 간 산술 연산이 가능합니다.
result = first_num + second_num
print(result)  # 사용자가 5와 7을 입력했다면, 출력은 "12"
```

만약 소수점(Float)이 필요한 경우라면, `int()` 대신 `float()`을 사용하면 됩니다. `str()`, `int()`, `float()` 등의 함수를 사용해 데이터 타입을 적재적소에 맞게끔 변환하는 것은 이러한 이유 때문입니다.


### 내장 함수 사용 연습

그럼 이제 내장함수 연습을 해 보겠습니다.

'프렌즈' 주요 등장인물 리스트 `main_cast = ["Monica", "Rachel", "Phoebe", "Joey", "Chandler", "Ross"]` 와 챈들러 정보 딕셔너리 `chandler_info = {"이름": "챈들러 빙", "직업": "데이터 처리 전문가 (불확실)", "파트너": "모니카"}` 가 있다고 가정해 봅시다. 데이터를 바탕으로 다음의 과정을 수행하는 코드를 작성해 보겠습니다.

1. `len()` 함수를 사용해서 `main_cast` 리스트에 몇 명의 등장인물이 있는지 출력한다.
2. `len()` 함수를 사용해서 챈들러의 이름("챈들러 빙")의 글자 수를 출력한다. (`chandler_info` 딕셔너리 활용)
3. 시즌 번호 3 (정수)을 `str()` 함수를 사용해서 문자열로 바꾼 뒤, `"시즌 "`과 합쳐서 `"시즌 3"`이라고 출력한다.
4. 평점 `"8.5"` (문자열)를 `float()` 함수를 사용해서 실수로 바꾸고, 그 타입을 출력한다.

```
# 예제 데이터 정의
main_cast = ["Monica", "Rachel", "Phoebe", "Joey", "Chandler", "Ross"]
chandler_info = {"이름": "챈들러 빙", "직업": "데이터 처리 전문가 (불확실)", "파트너": "모니카"}

# 1. 리스트 길이 구하기
num_cast = len(main_cast)
print(f"주요 등장인물 수: {num_cast}")

# 2. 문자열 길이 구하기 (딕셔너리 값 접근 후)
chandler_name = chandler_info["이름"]
name_length = len(chandler_name)
print(f"챈들러 이름 글자 수: {name_length}")

# 3. 정수를 문자열로 변환하여 합치기
season_num = 3
season_str = str(season_num)
print("시즌 " + season_str) # 문자열 + 문자열 결합

# 4. 문자열을 실수로 변환하고 타입 확인
rating_str = "8.5"
rating_float = float(rating_str)
print(f"문자열 '{rating_str}'를 실수로 변환: {rating_float}")
print(f"변환 후 타입: {type(rating_float)}")
```

**실행 및 설명**

* 결과 확인
    * 각 단계별 결과가 출력됩니다.
    * `len()` 함수는 리스트의 요소 개수나 문자열의 글자 수를 세는 데 사용됩니다.
    * `str()`, `int()`, `float()` 함수는 데이터 타입을 변환해야 할 때 유용합니다. 예를 들어 숫자와 문자열을 직접 더할 수는 없으므로, 숫자를 문자열로 변환(`str()`)한 후 이어 붙여야 합니다.
