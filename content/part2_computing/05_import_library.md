## 라이브러리 임포트

지금까지 우리는 파이썬의 기본 문법과 함수를 만드는 법을 배웠습니다. 구체적으로, 함수를 통해 코드를 재사용하고 구조화하는 방법을 배웠습니다. 

그런데 만약 우리가 하려는 작업을 누군가 이미 훌륭하게 구현해 놓았다면 어떨까요? 복잡한 수학 계산, 임의의 요소 선택, 날짜 처리 등을 위한 함수를 매번, 직접 만들 필요 없이, 다른 사람이 만들어 놓은 코드 묶음을 가져다 쓸 수 있다면 훨씬 효율적일 것입니다.

파이썬에서는 이렇게 특정 기능들과 관련된 코드 즉 함수, 변수 등을 미리 묶어 놓은 것을 '**모듈(Module)**' 또는 '**라이브러리(Library)**'라고 부릅니다. 마치 우리가 특정 주제를 깊이 연구할 때 관련 논문이나 전문 서적을 참고하는 것처럼, 파이썬 프로그래밍에서도 특정 작업을 할 때 관련된 라이브러리를 가져와서 사용하는 것이 일반적입니다.

* **라이브러리:** 특정 기능 관련 모듈들을 모아 놓은 더 큰 단위입니다.
    * 예시로, 수학 관련 기능 모음 `math`, 텍스트 분석 기능 모음 `nltk` 등이 있습니다.
* **모듈:** 파이썬 코드를 담고 있는 파일(`.py`) 하나를 의미하기도 합니다. 라이브러리는 여러 모듈로 구성될 수 있습니다.

라이브러리를 사용하면 다음과 같은 장점이 있습니다.

1.  **코드 재사용:** 이미 검증된 코드를 가져다 쓰므로 개발 시간을 단축하고 오류를 줄일 수 있습니다.
2.  **기능 확장:** 파이썬 기본 기능만으로는 하기 어려운 복잡한 작업(데이터 분석, 웹 개발, 게임 제작 등)을 가능하게 합니다.
3.  **표준화:** 널리 사용되는 라이브러리를 이용하면 다른 개발자들과 협업하기 용이합니다.

이 장에서는 파이썬에서 라이브러리(모듈)를 불러와 사용하는 기본적인 방법인 `import` 문에 대해 알아보겠습니다.

## 도구 상자 가져오기: `import` 문

파이썬에서 다른 사람이 만든 라이브러리, 모듈의 기능을 사용하려면, 먼저 `import` 라는 명령어를 사용하여 해당 라이브러리를 현재 작업 환경으로 불러와야 합니다. 이는 마치 작업에 필요한 도구 상자를 작업대 위에 올려놓는 것, 혹은 사용하고자 하는 프로그램을 실행시키는 것과 유사하며, import문의 기본 형식은 다음과 같습니다.

```
import 라이브러리이름
```

라이브러리를 import 한 후에는 `라이브러리이름.함수이름()` 또는 `라이브러리이름.변수이름` 형태로 해당 라이브러리 안의 기능(함수)이나 값(변수)을 사용할 수 있습니다. 점(`.`)은 '-의 안에 있는' 이라는 의미로 생각할 수 있으며, 앞서 언급한 경우 import한 라이브러리 안에 있는 여러 함수, 변수 중 특정 함수, 변수를 가져오는 것이 됩니다.

## 수학 도구 상자 활용하기 (`math` 라이브러리)

파이썬에는 기본적인 수학 계산을 위한 `math` 라이브러리가 내장되어 있습니다. 제곱근 계산, 원주율(π) 값 사용 등을 할 수 있습니다.

그럼 이제, 수학 계산을 위해 파이썬의 `math` 라이브러리를 사용해 봅시다. 

먼저 `import math` 코드로 math 라이브러리를 import하고, '프렌즈' 주요 등장인물은 6명인데, 이 숫자 6의 제곱근을 `math.sqrt()` 함수로 계산해서 `sqrt_cast_size` 변수에 저장하고, 설명과 함께 출력하는 코드를 실행시켜 봅시다. 이때 제곱근은 캐릭터 네트워크 분석에서 '거리' 개념과 관련될 수 있는데, 이후 네트워크 분석 부분에서 상술하도록 하겠습니다.

```
# math 라이브러리를 불러옵니다.
import math

# 주요 등장인물 수
num_main_cast = 6

# math 라이브러리의 sqrt() 함수를 사용하여 제곱근 계산
sqrt_cast_size = math.sqrt(num_main_cast)

# 계산 결과를 출력합니다.
print(f"'프렌즈' 주요 등장인물 수({num_main_cast})의 제곱근: {sqrt_cast_size}")
```

**실행 및 설명**

* 결과 확인
    * `'프렌즈' 주요 등장인물 수(6)의 제곱근: 2.449489742783178`
        * 소수점 이하 자릿수는 다를 수 있음에 주의하세요.
    * `import math`를 통해 `math` 라이브러리의 기능을 사용할 준비를 합니다.
    * 제곱근을 계산하기 위해 `math.sqrt(num_main_cast)`처럼 `라이브러리이름.함수이름()` 형식으로 함수를 호출했습니다.

`math` 라이브러리에는 원주율(π) 같은 중요한 수학 상수도 미리 정의되어 있습니다. 

이번에는 `math.pi`를 사용해서 원주율 값을 `'원주율(pi) 값:'` 설명과 함께 출력하는 코드를 실행시켜 봅시다. 이때, `math` 라이브러리는 이미 import되었다고 가정합니다.

```
# math 라이브러리는 이미 import 되었다고 가정합니다.
# import math  # 만약 이전 셀에서 import하지 않았다면 이 줄의 주석을 해제

# math 라이브러리의 pi 변수(원주율 값)를 사용
print(f"원주율(pi) 값: {math.pi}")
```

**실행 및 설명:**

*  결과 확인
    * `원주율(pi) 값: 3.141592653589793`
    * 원주율 값을 사용하기 위해 `math.pi`처럼 `라이브러리이름.변수이름` 형식으로 접근했습니다.


## 무작위 선택의 즐거움: `random` 라이브러리

때로는 프로그램에서 예측 불가능한 무작위(Random) 요소를 사용하고 싶을 때가 있습니다. 예를 들어, 여러 캐릭터 중 한 명을 임의로 선택하여 분석하거나, 여러 주제 중 하나를 무작위로 고르는 경우 등입니다. 

파이썬의 `random` 라이브러리가 이러한 기능을 제공하며, `random` 라이브러리에 내장된 주요 함수는 다음과 같습니다.

| 함수 | 역할 | 형식 | 예시 |
| :--- | :--- | :--- | :--- |
| `randint()` | 주어진 범위(**시작 숫자와 끝 숫자 포함**) 내의 **정수 난수** 하나를 무작위로 생성해 반환합니다. | `random.randint(a, b)` | **a** 이상 **b** 이하의 **정수** $N$을 반환하며, `randint(1, 3)`의 경우 **1, 2, 3** 중 하나를 반환합니다. |
| `choice()` | **리스트**에서 무작위 요소 하나를 선택하여 반환합니다. | `random.choice(리스트)` | `random.choice(["로스", "레이첼", "조이"])`는 **셋 중 한 명**을 무작위로 반환합니다. |
| `shuffle()` | **리스트 요소 순서**를 무작위로 섞습니다. | `random.shuffle(리스트)` | `friends=["모니카","챈들러","피비"]`, `random.shuffle(friends)`를 실행하면 **friends 리스트의 순서**가 무작위로 변경되며, 이러한 변화는 **원본 리스트 자체의 변화**로 이어집니다. |

정수 난수를 생성하는 대표적인 함수로 `randint()` 외에, 범위를 지정해 정수 난수를 생성할 수 있는 `randrange()`가 있습니다. 두 함수는 종료 값의 포함 여부에서 중요한 차이가 있으니 각별히 주의해야 합니다.

| 함수 | 형식 | 종료 값 포함 여부 | 예시: random.randint(1, 10) |
| :--- | :--- | :--- | :--- |
| `randint()` | `randint(a, b)` | 포함 (**a ≤ N ≤ b**) | 1부터 10까지의 정수 중 하나 (**10 포함**) |
| `randrange()` | `randrange(a, b)` | 미포함 (**a ≤ N < b**) | 1부터 9까지의 정수 중 하나 (**10 미포함**) |

`randint(a, b)`는 a 이상 b 이하의 정수를 반환하므로, 범위를 직관적으로 설정하고 싶을 때 유용합니다. 반면 `randrange(a, b)`는 `range()` 함수와 마찬가지로 시작 값은 포함하고 끝 값은 제외하는 파이썬의 일반적인 범위 규칙을 따릅니다.

### 분석 대상 캐릭터 무작위 선택하기

그럼 `random` 라이브러리를 활용해, 인물 분석 시 무작위로 대상을 선정하는 방법을 연습해 보겠습니다. 

먼저 `import random`으로 `random` 라이브러리를 import하고, '프렌즈' 주요 등장인물 리스트 `main_cast = ["Monica", "Rachel", "Phoebe", "Joey", "Chandler", "Ross"]` 에서 `random.choice()` 함수를 사용하여 무작위로 한 명의 캐릭터를 선택한 뒤, `'이번 분석 대상 캐릭터: [선택된 캐릭터 이름]'` 형식으로 출력하는 코드를 작성해 실행시켜 봅시다.

```
# random 라이브러리를 불러옵니다.
import random

# 주요 등장인물 리스트 정의
main_cast = ["Monica", "Rachel", "Phoebe", "Joey", "Chandler", "Ross"]

# random.choice() 함수를 사용하여 리스트에서 무작위로 하나 선택
chosen_character = random.choice(main_cast)

# 선택된 캐릭터 이름을 출력합니다.
print(f"이번 분석 대상 캐릭터: {chosen_character}")
```

**실행 및 설명**

* 결과 확인
    * 코드를 실행할 때마다 6명의 주인공 중 한 명의 이름이 무작위로 출력됩니다.
    * `import random`으로 라이브러리를 불러왔습니다.
    * `random.choice(main_cast)`를 통해 `main_cast` 리스트의 요소 중 하나가 무작위로 선택되어 `chosen_character` 변수에 저장됩니다. 
        * 이는 문학 작품 분석에서 무작위 샘플링을 하거나, 토론 주제를 정하는 등 다양한 상황에 활용될 수 있습니다.

### 가상 주제 언급 횟수 생성하기

이번에는 `random` 라이브러리 사용해, 1부터 10 사이의 정수 하나 무작위로 뽑아서 출력해 보겠습니다.

어떤 에피소드에서 특정 주제(예: '가족')가 몇 번 언급될지 예측하기 어렵다고 가정해 보겠습니다. `random` 라이브러리의 `randint()` 함수를 사용해 가상의 주제 언급 횟수에 대응되는 1부터 10 사이의 정수 하나를 무작위로 생성하고, `'이번 에피소드 가상 주제 언급 횟수: [숫자]'` 형식으로 출력하는 코드를 작성하고 실행시켜 봅시다. 이때 `random` 라이브러리는 이미 import 되었다고 가정하겠습니다.

```
# random 라이브러리는 이미 import 되었다고 가정합니다.
# import random # 만약 이전 셀에서 import 안했다면 이 줄 필요

# 1부터 10까지의 범위에서 무작위 정수 생성
# randint(a, b)는 a <= N <= b 인 정수 N을 반환합니다.
random_mentions = random.randint(1, 10)

# 생성된 무작위 횟수를 출력합니다.
print(f"이번 에피소드 가상 주제 언급 횟수: {random_mentions}")
```

**실행 및 설명**

* 결과 확인
    * 실행할 때마다 1부터 10 사이의 정수가 무작위로 출력됩니다.
    * `random.randint(1, 10)` 함수는 1과 10을 포함한 범위 내에서 임의의 정수를 생성합니다. 
        * 이는 시뮬레이션이나 무작위성을 필요로 하는 분석에 활용될 수 있습니다.
