컴퓨터는 본질적으로 데이터를 처리하는 기계입니다. 우리가 컴퓨터에게 의미 있는 작업을 시키려면, 먼저 다루어야 할 정보, 즉 **데이터(Data)**를 컴퓨터가 이해하고 저장할 수 있는 형태로 표현해야 합니다. 이 과정에서 핵심적인 역할을 하는 것이 바로 **변수(Variable)**와 **데이터 타입(Data Type)**입니다.

#### 변수(Variable): 데이터에 이름표 붙이기

**변수**는 데이터에 붙이는 **이름표** 또는 데이터를 담는 **상자**와 같습니다. 예를 들어 '프렌즈'의 명장면 중 하나인 "We were on a break!"라는 로스의 대사를 컴퓨터에 저장한다고 생각해 봅시다. 이 데이터를 그냥 저장하면 나중에 필요할 때 다시 찾아 쓰기가 매우 번거롭습니다. 그래서 우리는 이 데이터에 `ross_quote`와 같은 식별 가능한 이름표를 붙여줍니다. 이렇게 하면 나중에 `ross_quote`라는 이름표만 부르면 컴퓨터는 "We were on a break!"라는 데이터를 즉시 가져올 수 있습니다.

변수 이름은 우리가 이해하기 쉽도록 의미 있는 단어를 사용하는 것이 좋습니다. 파이썬에서는 변수 이름을 지을 때 몇 가지 규칙이 있습니다.

* 영문자, 숫자, 밑줄(`_`)을 사용할 수 있습니다.
* 숫자로 시작할 수는 없습니다. (예: `1st_character`는 안됨)
* 띄어쓰기는 사용할 수 없습니다. 대신 밑줄(`_`)을 사용합니다. (예: `main character` 대신 `main_character`)
* 대소문자를 구분합니다. (`myVariable`과 `myvariable`은 다른 변수입니다.)
* 파이썬이 이미 특별한 의미로 사용하고 있는 단어(예: `if`, `for`, `class` 등, 이를 '예약어'라고 합니다)는 변수 이름으로 사용할 수 없습니다.

변수에 데이터를 저장하는 과정을 **할당(Assignment)**이라고 하며, 등호(`=`) 기호를 사용합니다. `변수이름 = 데이터` 형식으로 작성하면, 오른쪽의 데이터가 왼쪽의 변수 이름표에 연결(저장)됩니다.

#### 데이터 타입(Data Type): 데이터의 종류 구분하기

**데이터 타입**은 말 그대로 데이터의 **종류**를 의미합니다. '레이첼 그린'처럼 글자로 이루어진 데이터도 있고, '프렌즈'의 총 시즌 수인 10처럼 소수점 없는 숫자로 된 데이터도 있으며, 에피소드 평점 8.4처럼 소수점이 있는 숫자도 있습니다. 또한, 여러 데이터를 순서대로 묶어 놓은 목록 형태의 데이터나, '이름: 레이첼, 직업: 패션'처럼 짝을 이룬 정보를 저장하는 데이터도 있습니다. '맞다/틀리다'를 나타내는 데이터 종류도 중요합니다.

컴퓨터는 데이터의 종류에 따라 처리하는 방식(예: 숫자끼리는 더할 수 있지만, 글자끼리는 더하는 것이 아니라 이어 붙임)이 다르기 때문에, 어떤 종류의 데이터인지 명확히 구분하는 것이 매우 중요합니다. 파이썬은 다양한 데이터 타입을 가지고 있지만, 이 장에서는 가장 기본적이고 자주 사용되는 여섯 가지 종류를 '프렌즈' 예시와 함께 AI 코드 생성을 통해 알아보겠습니다.

---

**글자 데이터 다루기 (문자열, String)**

컴퓨터 과학에서 글자, 단어, 문장과 같이 인간의 언어를 표현하는 텍스트 데이터를 통칭하여 **문자열(String)**이라고 부릅니다. 파이썬에서는 문자열 데이터를 코드 내에서 표현할 때, 데이터의 시작과 끝을 명확히 구분하기 위해 **따옴표(`"`)** 또는 **작은따옴표(`'`)**로 감싸줍니다.

* **간단 프롬프트 1:**
    > "`character_name_monica` 변수에 '모니카 겔러' 저장하고, 값과 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 1:**
    > "미국 드라마 '프렌즈'의 주인공 중 한 명인 '모니카 겔러'의 이름을 `character_name_monica` 라는 변수에 저장하고 싶어. 이 변수에 저장된 이름과 이 정보가 어떤 종류(데이터 타입)인지 화면에 보여주는 파이썬 코드를 AI로 만들어 줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # '모니카 겔러'라는 문자열 데이터를 character_name_monica 변수에 저장합니다.
    character_name_monica = "모니카 겔러" # 또는 'Monica Geller'

    # 변수에 저장된 값을 출력합니다. f-string을 사용하면 변수와 텍스트를 함께 출력하기 편리합니다.
    print(f"선택된 캐릭터 이름: {character_name_monica}")

    # 변수에 저장된 데이터의 종류(타입)를 확인하고 출력합니다.
    print(f"이름 정보의 종류: {type(character_name_monica)}")
```
* **실행 및 설명:**
    * 결과 확인: `선택된 캐릭터 이름: 모니카 겔러`, `이름 정보의 종류: <class 'str'>`
    * `=` 기호는 오른쪽의 `"모니카 겔러"`라는 문자열 값을 왼쪽의 `character_name_monica`라는 변수에 할당(저장)하는 역할을 합니다.
    * `type()` 함수는 괄호 안의 변수나 값이 어떤 데이터 타입인지 알려줍니다. `<class 'str'>`은 이 데이터가 파이썬에서 문자열(String) 타입으로 인식된다는 것을 의미합니다.

---

**소수점 없는 숫자 다루기 (정수, Integer)**

10, 24, 0, -5처럼 소수점 이하 부분이 없는 딱 떨어지는 숫자를 컴퓨터 과학에서는 **정수(Integer)**라고 부릅니다. 파이썬에서는 정수를 코드에 직접 쓸 때 따옴표를 사용하지 않고 숫자만 그대로 적습니다.

* **간단 프롬프트 2:**
    > "`total_seasons` 변수에 숫자 10 저장하고, 값과 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 2:**
    > "'프렌즈'는 총 10개의 시즌으로 완결되었어. 숫자 10을 `total_seasons` 라는 변수에 저장하고, 저장된 값과 이것이 어떤 종류의 데이터인지 화면에 보여주는 코드를 만들어줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # 총 시즌 수 10을 total_seasons 변수에 저장합니다. 따옴표 없이 숫자만 사용합니다.
    total_seasons = 10

    # 변수 값과 타입을 출력합니다.
    print(f"프렌즈 총 시즌 수: {total_seasons}")
    print(f"시즌 수 정보 종류: {type(total_seasons)}")
```
* **실행 및 설명:**
    * 결과 확인: `프렌즈 총 시즌 수: 10`, `시즌 수 정보 종류: <class 'int'>`
    * 숫자 10에는 따옴표가 없습니다. 파이썬은 이를 정수(Integer)로 인식하며, `type()` 함수는 `<class 'int'>`를 반환합니다.
    * **중요:** 숫자 10과 문자열 "10"은 컴퓨터에게 완전히 다른 데이터입니다. 숫자 10은 계산이 가능하지만, 문자열 "10"은 글자로 취급됩니다.

---

**소수점 있는 숫자 다루기 (실수, Float)**

3.14, 22.5, -0.8, 10.0처럼 소수점 이하 부분을 포함하는 숫자를 컴퓨터 과학에서는 **실수(Floating-point number)**라고 하며, 파이썬에서는 줄여서 **Float** 타입이라고 부릅니다. 정수와 마찬가지로 코드에 직접 쓸 때는 따옴표 없이 숫자 그대로 사용합니다.

* **간단 프롬프트 3:**
    > "`episode1_rating` 변수에 숫자 8.4 저장하고, 값과 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 3:**
    > "'프렌즈' 첫 번째 에피소드의 IMDb 평점은 10점 만점에 8.4점이라고 가정해보자. 숫자 8.4를 `episode1_rating` 변수에 저장하고, 그 값과 데이터 타입을 출력하는 코드를 만들어줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # 첫 에피소드 평점 8.4를 episode1_rating 변수에 저장합니다.
    episode1_rating = 8.4

    # 변수 값과 타입을 출력합니다.
    print(f"에피소드 1 평점: {episode1_rating}")
    print(f"평점 정보 종류: {type(episode1_rating)}")
```
* **실행 및 설명:**
    * 결과 확인: `에피소드 1 평점: 8.4`, `평점 정보 종류: <class 'float'>`
    * 소수점이 있는 숫자 8.4는 실수(Float) 타입으로 인식되며, `type()` 함수는 `<class 'float'>`를 반환합니다.

---

**여러 데이터를 순서대로 묶기 (리스트, List)**

여러 개의 데이터를 하나의 묶음으로, 넣은 순서대로 관리하고 싶을 때 **리스트(List)**를 사용합니다. 리스트는 **대괄호(`[]`)** 안에 각 데이터 항목(요소, element)을 넣고, 항목 사이를 **쉼표(`,`)**로 구분하여 만듭니다. 리스트 안에는 문자열, 숫자 등 다양한 종류의 데이터를 섞어서 넣을 수도 있습니다.

* **간단 프롬프트 4:**
    > "`main_cast` 변수에 '프렌즈' 주인공 6명 이름 리스트 저장하고, 리스트 전체와 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 4:**
    > "'프렌즈'의 주요 등장인물 6명 (모니카, 레이첼, 피비, 조이, 챈들러, 로스)의 이름을 순서대로 담은 목록(리스트)을 만들어서 `main_cast` 라는 변수에 저장해줘. 그리고 이 목록 전체와 이것이 어떤 종류의 정보인지 화면에 보여주는 코드를 만들어줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # 주요 등장인물 이름을 리스트로 만들어 main_cast 변수에 저장합니다.
    main_cast = ["모니카", "레이첼", "피비", "조이", "챈들러", "로스"] # 또는 영문 이름

    # 변수에 저장된 리스트 전체를 출력합니다.
    print("주요 등장인물 목록:", main_cast)

    # 변수에 저장된 데이터의 타입(리스트)을 확인하고 출력합니다.
    print("등장인물 목록 정보 종류:", type(main_cast))
```
* **실행 및 설명:**
    * 결과 확인: `주요 등장인물 목록: ['모니카', '레이첼', '피비', '조이', '챈들러', '로스']`, `등장인물 목록 정보 종류: <class 'list'>`
    * 대괄호 `[]` 안에 여러 개의 문자열(`str`)이 쉼표로 구분되어 순서대로 들어있는 것을 볼 수 있습니다. 이것이 바로 리스트(List)이며, 타입은 `list`로 표시됩니다.
    * 리스트의 특징은 **순서가 있다**는 점과, 안에 있는 값을 나중에 **바꿀 수 있다**(mutable)는 점입니다.

---

**이름표 붙은 정보 묶기 (딕셔너리, Dictionary)**

리스트는 순서대로 데이터를 나열하지만, 각 데이터가 무엇을 의미하는지 알려주지는 않습니다. 만약 '이름은 레이첼', '직업은 패션'처럼 **이름표(Key)와 값(Value)을 짝지어** 정보를 저장하고 싶다면 **딕셔너리(Dictionary)**를 사용합니다. 딕셔너리는 **중괄호(`{}`)** 안에 `Key: Value` 쌍들을 넣고, 각 쌍 사이를 **쉼표(`,`)**로 구분하여 만듭니다. Key는 보통 문자열이나 숫자를 사용하며, Value에는 어떤 데이터 타입이든 올 수 있습니다.

* **간단 프롬프트 5:**
    > "`chandler_info` 변수에 챈들러 정보 (이름: '챈들러 빙', 직업: '데이터 처리 전문가 (불확실)', 파트너: '모니카') 딕셔너리 저장하고, 전체와 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 5:**
    > "챈들러의 정보를 담는 딕셔너리를 만들고 싶어. '이름'이라는 이름표에는 '챈들러 빙'을, '직업'이라는 이름표에는 '데이터 처리 전문가 (불확실)'를, '파트너' 이름표에는 '모니카'를 값으로 넣어서 `chandler_info` 변수에 저장하고, 이 딕셔너리 전체와 타입을 출력하는 코드를 만들어줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # 챈들러 정보를 Key:Value 쌍으로 딕셔너리에 저장합니다.
    chandler_info = {
        "이름": "챈들러 빙", # 또는 "Chandler Bing"
        "직업": "데이터 처리 전문가 (불확실)", # 또는 "IT Procurement Manager (Transponster?)"
        "파트너": "모니카" # 또는 "Monica"
    }

    # 변수에 저장된 딕셔너리 전체를 출력합니다.
    print("챈들러 정보:", chandler_info)

    # 변수에 저장된 데이터의 타입(딕셔너리)을 확인하고 출력합니다.
    print("정보 묶음 종류:", type(chandler_info))
```
* **실행 및 설명:**
    * 결과 확인: `챈들러 정보: {'이름': '챈들러 빙', '직업': '데이터 처리 전문가 (불확실)', '파트너': '모니카'}`, `정보 묶음 종류: <class 'dict'>`
    * 중괄호 `{}` 안에 `Key: Value` 쌍들이 들어있는 것을 볼 수 있습니다. 이것이 딕셔너리(Dictionary)이며, 타입은 `dict`로 표시됩니다.
    * 딕셔너리는 리스트와 달리 **순서가 중요하지 않습니다.** 대신, 고유한 Key(이름표)를 통해 원하는 Value(값)에 직접 접근할 수 있습니다. (예: `chandler_info['직업']`은 '데이터 처리 전문가 (불확실)' 값을 가져옵니다.)

**'맞다/틀리다' 표현하기 (불리언, Boolean)**

프로그래밍에서는 어떤 조건이 '맞다' 또는 '틀리다'를 표현해야 할 때가 많습니다. 이때 사용하는 것이 **불리언(Boolean)** 데이터 타입입니다. 불리언은 오직 두 가지 값, **`True`**(참, 맞다)와 **`False`**(거짓, 틀리다) 만을 가집니다. 파이썬에서는 첫 글자를 대문자로 써야 합니다.

* **간단 프롬프트 6:**
    > "`ross_rachel_were_couple` 변수에 True 값 저장하고, 값과 타입 출력하는 코드 만들어줘."
* **상세 프롬프트 6:**
    > "'프렌즈'에서 로스와 레이첼이 연인 관계였던 적이 있다는 사실은 '맞다(True)'야. 이 값을 `ross_rachel_were_couple` 변수에 저장하고, 값과 타입을 출력하는 코드를 만들어줘."
* **(AI 생성 코드 예상 및 함께 검토):**
```python
    # 로스와 레이첼이 연인이었는지 여부를 불리언 값으로 저장합니다.
    ross_rachel_were_couple = True

    # 변수 값과 타입을 출력합니다.
    print(f"로스와 레이첼은 연인이었나?: {ross_rachel_were_couple}")
    print(f"참/거짓 정보 종류: {type(ross_rachel_were_couple)}")
```
* **실행 및 설명:**
    * 결과 확인: `로스와 레이첼은 연인이었나?: True`, `참/거짓 정보 종류: <class 'bool'>`
    * `True`는 불리언(Boolean) 타입의 값이며, 타입은 `bool`로 표시됩니다.
    * 비교 연산자(예: `5 > 3`)의 결과도 불리언 값(`True` 또는 `False`)으로 나옵니다. 불리언은 나중에 배울 조건문에서 매우 중요하게 사용됩니다.
