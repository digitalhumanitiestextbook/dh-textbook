## 4.3. 전체 인물 네트워크의 기본 통계량 분석 및 해석

이제 우리 손에 "프렌즈"의 사회적 우주 전체를 담고 있는, 살아있는 네트워크 객체 `G_directed`가 쥐어졌습니다. 이제 우리는 이 객체에게 질문을 던져 그 속성을 파악할 수 있습니다. 마치 완성된 건축물의 제원을 측정하듯, 우리는 네트워크의 크기, 연결 밀도, 전체적인 구조적 특성을 나타내는 몇 가지 기본적인 "**통계량(statistics)**"을 계산해 볼 것입니다.

이러한 기본 통계량들은 우리가 더 복잡한 분석(예: 중심성 분석)으로 나아가기 전에, 우리 네트워크의 "건강 상태"를 진단하고 전반적인 성격을 파악하는 데 필수적인 지표입니다. 이 숫자들은 그 자체로도 "프렌즈의 전체 세계는 얼마나 끈끈한 집단인가?", "핵심 그룹과 주변부는 어떻게 연결되어 있는가?" 와 같은 흥미로운 질문에 대한 첫 번째 답을 제공해 줄 것입니다.

---

### 1단계: 네트워크의 기본 규모 확인 (노드, 엣지 수)

가장 먼저 확인할 것은 우리 네트워크의 규모입니다. 분석 대상이 되는 "배우"는 몇 명이며, 그들 사이의 "관계"는 총 몇 개인지를 확인합니다.

(앞선 코드에 이어서, 아래 코드를 다음 셀에 복사하여 실행하세요. 4.2절에서 생성된 `G_directed` 객체가 필요합니다.)

#### [코드]
```
# (4.2절에서 생성한 G_directed 객체가 필요합니다)

# 그래프의 노드 수 출력
num_nodes = G_directed.number_of_nodes()
print(f"네트워크의 노드 수: {num_nodes}")

# 그래프의 엣지 수 출력
num_edges = G_directed.number_of_edges()
print(f"네트워크의 엣지 수: {num_edges}")
```

#### [실행 결과]
네트워크의 노드 수: 417
네트워크의 엣지 수: 2962

#### [결과 해석]
실행 결과, 우리의 "프렌즈" 세계는 총 **417명의 등장인물 (노드)**과, 그들을 잇는 **2,962개의 방향성 관계 (엣지)**로 구성되어 있음을 확인했습니다. 이는 결코 작지 않은 규모의 사회적 네트워크이며, 이 숫자들은 앞으로 우리가 수행할 모든 분석의 가장 기본적인 규모와 범위를 설정해 줍니다.

### 2단계: 네트워크의 연결성 분석 (밀도, 연결 요소)

다음으로, 이 417명의 사회가 얼마나 빽빽하게, 그리고 하나로 잘 연결되어 있는지를 측정해 보겠습니다.

(앞선 코드에 이어서, 아래 코드를 다음 셀에 복사하여 실행하세요.)

#### [코드]
```
# 네트워크의 밀도를 계산합니다.
density = nx.density(G_directed)
print(f"네트워크 밀도: {density:.3f}")

# 방향성 그래프의 연결성을 확인합니다.
# "약한 연결"은 방향을 무시했을 때, 모든 노드가 하나로 연결된 상태를 의미합니다.
is_weakly_connected = nx.is_weakly_connected(G_directed)
print(f"네트워크는 약하게 연결되어 있는가? {is_weakly_connected}")

if not is_weakly_connected:
    # 만약 약하게 연결되어 있지 않다면, 몇 개의 "섬"으로 나뉘어 있는지 확인합니다.
    num_components = nx.number_weakly_connected_components(G_directed)
    print(f"연결 요소(섬)의 개수: {num_components}")
    
    # 가장 큰 "섬"에 몇 명이 속해있는지 확인합니다.
    largest_cc = max(nx.weakly_connected_components(G_directed), key=len)
    print(f"가장 큰 연결 요소의 크기: {len(largest_cc)}명")
```


#### [실행 결과]
네트워크 밀도: 0.017
네트워크는 약하게 연결되어 있는가? False
연결 요소(섬)의 개수: 11
가장 큰 연결 요소의 크기: 406명

#### [결과 해석]
- **네트워크 밀도 (Density):** 밀도는 **0.017**이라는 매우 낮은 수치를 보여줍니다. 이는 417명의 인물 사이에서 이론적으로 형성 가능한 모든 방향성 관계 중, 실제로는 약 1.7%만이 존재한다는 의미입니다. 이는 "프렌즈"의 전체 사회가 결코 모두가 서로를 아는 긴밀한 공동체가 아니라, 대부분의 사람들이 서로 관계를 맺지 않는 **매우 희소한(sparse) 네트워크**임을 보여줍니다. 이는 소수의 핵심 그룹이 빽빽한 관계를 맺고, 수많은 주변 인물들이 드문드문 연결된 "핵심-주변부" 구조를 가졌을 것이라는 우리의 가설을 뒷받침합니다.

- **연결 요소 (Connected Components):** 분석 결과, 이 네트워크는 **총 11개의 분리된** "**섬**"으로 이루어져 있습니다. 이는 "프렌즈" 세계의 모든 인물이 하나의 거대한 관계망 안에 있는 것이 아니라, 서로 전혀 연결되지 않는 몇몇 소그룹이나 고립된 개인들이 존재함을 의미합니다. 하지만 가장 큰 섬에 **406명**이 속해 있다는 것은, 대부분의 인물(약 97%)은 주인공 6명을 중심으로 한 거대한 하나의 관계망 안에 포함되어 있음을 시사합니다.

### 3단계: 네트워크의 거리 측정 (지름, 평균 최단 경로)

네트워크가 여러 개의 섬으로 나뉘어 있으므로, 전체 네트워크의 지름이나 평균 거리를 측정하는 것은 의미가 없습니다. 따라서 우리는 가장 큰 섬(거대 연결 요소)에 속한 406명의 사람들 사이에서, 거리가 얼마나 되는지를 측정해 보겠습니다.

(앞선 코드에 이어서, 아래 코드를 다음 셀에 복사하여 실행하세요.)

#### [코드]
```
# 약하게 연결된 경우에만 실행됩니다.
if not nx.is_weakly_connected(G_directed):
    # 가장 큰 컴포넌트의 노드 목록을 가져옵니다.
    largest_cc = max(nx.weakly_connected_components(G_directed), key=len)
    # 해당 노드들로만 구성된 하위 그래프(subgraph)를 생성합니다.
    subgraph = G_directed.subgraph(largest_cc)
    
    # 방향성을 무시하고 거리를 계산하기 위해 무방향 그래프로 변환합니다.
    G_undirected_subgraph = subgraph.to_undirected()

    # 하위 그래프의 지름과 평균 최단 경로 길이를 계산합니다.
    diameter = nx.diameter(G_undirected_subgraph)
    print(f"\n네트워크 지름 (가장 큰 컴포넌트 기준): {diameter}")
    
    avg_path_length = nx.average_shortest_path_length(G_undirected_subgraph)
    print(f"평균 최단 경로 길이 (가장 큰 컴포넌트 기준): {avg_path_length:.3f}")
```


#### [실행 결과]
네트워크 지름 (가장 큰 컴포넌트 기준): 5
평균 최단 경로 길이 (가장 큰 컴포넌트 기준): 2.273

#### [결과 해석]
- **네트워크 지름 (Diameter):** 가장 큰 관계망 내에서, 가장 멀리 떨어져 있는 두 사람도 최대 **5단계**만 거치면 모두 연결될 수 있다는 것을 의미합니다. 400명이 넘는 거대한 네트워크임에도 불구하고, 5단계만 거치면 모두 연결된다는 것은 매우 흥미로운 결과입니다.

- **평균 최단 경로 길이 (Average Shortest Path Length):** 이 관계망에 속한 임의의 두 사람을 선택했을 때, 그들 사이에는 평균적으로 약 **2.27단계**가 존재한다는 뜻입니다. 즉, "내 친구의 친구" 정도만 거치면 대부분의 사람에게 도달할 수 있다는 의미입니다.

이 두 지표는, 비록 전체 네트워크의 밀도는 매우 낮지만, 그 내부는 주인공 6명이라는 강력한 "허브(Hub)" 덕분에 매우 효율적으로 연결된 "**작은 세상(small-world)**"의 특징을 강하게 보여줍니다.

### 🤖 AI와 함께 탐색하기: 통계 숫자에서 사회적 의미 읽어내기

**학습 목표:** 추상적인 네트워크 통계 수치를 보고, 그 숫자가 현실 세계에서 어떤 사회적 구조나 분위기를 의미하는지 상상하고 해석하는 능력을 기릅니다.

**간단 프롬프트:**
`네트워크의 밀도(density)가 0.01인 그룹과 0.9인 그룹은 어떻게 다를까?`

**상세 프롬프트:**
`내가 분석 중인 어떤 회사의 부서 내 이메일 네트워크에 대한 정보가 아래와 같아.`
`- 노드 수: 20명`
`- 엣지 수: 30개`
`- 네트워크 밀도: 약 0.08`
`- 연결 요소: 3개`
`이 통계 수치들을 바탕으로, 이 부서의 소통 구조와 사회적 분위기가 어떠할지 구체적으로 묘사해줘. 어떤 잠재적인 문제점이 있을 수 있을까?`

**인문학적 프롬프트:**
`"평균 최단 경로 길이"가 짧은 "작은 세상(small-world)" 네트워크는 종종 "효율적인" 또는 "응집력 있는" 집단으로 긍정적으로 평가돼. 하지만 이러한 구조가 갖는 어두운 면은 없을까? 예를 들어, 집단 내부의 정보가 너무 빨리 공유될 때 발생할 수 있는 부작용(예: 집단사고, 프라이버시 부재, 외부 의견 배척)은 무엇일지, "프렌즈"의 주인공 그룹을 예시로 들어 비판적으로 논해줘. "가까움"이 항상 "좋음"을 의미하지는 않는다는 점을 설명해줘.`
