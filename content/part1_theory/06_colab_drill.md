## Colab 실습 - 기본 기능 익히기

이 장에서는 앞서 소개된 Google Colab의 기본 기능들을 직접 실습하며 익혀보겠습니다. 실제 코드를 실행하고, 텍스트를 꾸미고, 파일을 다루는 기초적인 방법을 체험하며 Colab 환경과 친숙해지는 것을 목표로 합니다.

## 코드 셀 사용하기: "Hello, World!" 출력

프로그래밍 언어를 배울 때 가장 먼저 해보는 관례적인 작업은 화면에 "Hello, World!"라는 문구를 출력하는 것입니다. Colab의 코드 셀을 이용하여 이를 실행해 봅시다.

1.  Colab 노트북에서 `+ 코드` 버튼을 클릭하여 새로운 코드 셀을 추가합니다.
2.  생성된 코드 셀 안에 다음 파이썬 코드를 정확히 입력합니다.

    ```python
    print("Hello, World!")
    ```
    
   
3.  코드 셀 왼쪽의 '**재생**' 버튼(▶)을 클릭하거나, 셀을 선택한 상태에서 `Shift + Enter` 키를 누릅니다.
4.  셀 바로 아래에 다음과 같이 결과가 출력되는지 확인합니다.

    ```
    Hello, World!
    ```

`print()`는 파이썬에서 괄호 안의 내용을 화면에 보여달라고 컴퓨터에게 지시하는 명령어(함수)입니다. 따옴표(`"`)로 감싸진 "Hello, World!"는 **문자열** 데이터입니다. 성공적으로 실행되었다면, 여러분은 Colab에서 첫 파이썬 코드를 실행한 것입니다!

## 텍스트 셀 활용하기: 마크다운과 목차

코드뿐만 아니라 설명을 곁들이거나 문서 구조를 만드는 것도 중요합니다. Colab의 텍스트 셀은 **'마크다운(Markdown)'**이라는 간단한 서식 문법을 지원하여 이를 가능하게 합니다.

1.  노트북 상단의 `+ 텍스트` 버튼을 클릭하여 새로운 텍스트 셀을 추가합니다.
2.  생성된 텍스트 셀을 더블 클릭하여 편집 모드로 들어간 후, 다음 내용을 입력해 보세요.

```
    # 이것은 가장 큰 제목입니다 (h1)

    ## 이것은 중간 크기 제목입니다 (h2)

    ### 이것은 작은 제목입니다 (h3)

    일반 텍스트입니다. *이탤릭체* 와 **볼드체** 를 사용할 수 있습니다.

    * 순서 없는 목록 1
    * 순서 없는 목록 2

    1. 순서 있는 목록 1
    2. 순서 있는 목록 2

    [이것은 링크입니다](https://www.google.com)

    ---
    구분선입니다.
```
    


* **목차 생성:**

Colab은 텍스트 셀에 작성된 제목 (`#`, `##`, `###` 등)을 기반으로 자동으로 목차를 생성하는 편리한 기능을 제공합니다.

1.  노트북 화면 왼쪽의 사이드바에서 **목차 아이콘**(세 줄 모양)을 클릭합니다.
2.  텍스트 셀에 작성한 제목들이 계층 구조로 나타나는 것을 볼 수 있습니다.
3.  목차의 항목을 클릭하면 해당 제목이 있는 셀로 바로 이동합니다. 노트북 내용이 길어질 때 매우 유용합니다.

## 간단한 파일 불러오기

Colab에서 외부 데이터를 사용하려면 파일을 Colab 환경으로 가져와야 합니다. 가장 간단한 방법은 직접 업로드하는 것입니다.

1.  노트북 화면 왼쪽의 사이드바에서 **폴더 아이콘**을 클릭하여 파일 탐색기를 엽니다.
2.  파일 탐색기 상단의 '**파일 업로드**' 아이콘(위쪽 화살표 모양)을 클릭합니다.
3.  컴퓨터에서 업로드할 파일을 선택합니다. (예: 간단한 텍스트 파일 `.txt`)
4.  업로드가 완료되면 파일 탐색기 목록에 해당 파일이 나타납니다.

**주의**: 이 방법으로 업로드된 파일은 Colab 런타임이 종료되면 **사라집니다**. 영구적으로 파일을 사용하려면 구글 드라이브를 연동하는 방법(추후 학습)을 사용하는 것이 좋습니다.

## 런타임(Runtime)이란?

Colab에서 코드를 실행하면, 실제로는 구글의 서버에 있는 가상의 컴퓨터 자원을 빌려 쓰는 것입니다. 이 가상의 실행 환경을 **'런타임(Runtime)'**이라고 부릅니다.

* **런타임 연결**: 노트북을 열고 코드 셀을 처음 실행하면 자동으로 런타임에 연결됩니다. 화면 오른쪽 상단에서 연결 상태(RAM, 디스크 사용량 등)를 확인할 수 있습니다.
* **런타임 유형**: 메뉴의 **런타임 > 런타임 유형 변경**에서 하드웨어 가속기 사용 여부를 선택할 수 있습니다. 무료 버전에서는 주로 다음 옵션이 제공됩니다(가용성은 변동될 수 있습니다):
    * `None`: 기본 설정으로, 일반적인 **CPU**(Central Processing Unit)를 사용합니다. 대부분의 기초적인 코드 실행 및 데이터 분석 작업에 적합합니다.
    * `T4 GPU`: **GPU**(Graphics Processing Unit)는 병렬 처리에 특화되어 있어 딥러닝 모델 학습이나 대규모 계산 작업 속도를 크게 향상시킬 수 있습니다. 무료 버전에서는 T4와 같은 특정 모델의 GPU가 제한적으로 제공될 수 있습니다.
    * `TPU`: **TPU**(Tensor Processing Unit)는 구글이 딥러닝 계산(특히 TensorFlow)에 최적화하여 개발한 하드웨어입니다. 특정 유형의 대규모 머신러닝 작업에서 GPU보다 더 빠른 성능을 보일 수 있습니다. 무료 버전에서 사용 가능 여부는 제한적일 수 있습니다.
* **런타임 재시작/종료**: 코드가 꼬이거나 메모리 문제가 발생했을 때, 런타임 메뉴에서 '**런타임 재시작**', '**런타임 연결 해제 및 삭제**' 등을 통해 환경을 초기화할 수 있습니다. 런타임을 재시작하거나 종료하면 설치했던 라이브러리나 업로드했던 파일, 변수 값 등이 **모두 사라지므로 주의**해야 합니다.
* **세션 유지**: Colab은 일정 시간(무료 버전 기준 약 12시간 최대, 비활성 시 더 짧음) 동안만 런타임을 유지합니다. 장시간 작업 시 중간중간 저장하고, 필요시 런타임을 다시 연결해야 합니다.

## 생성 AI 활용하여 코드 만들기 및 이해하기

Colab은 내장된 생성 AI(**Gemini** 모델 기반)를 활용하여 코딩 작업을 도와주는 편리한 기능을 제공합니다. 크게 두 가지 방식으로 사용할 수 있습니다.

### 방법 1: 코드 셀에서 직접 생성 및 설명 요청하기

코드 셀 상단에 나타나는 버튼들을 활용하면 코드 생성뿐만 아니라 기존 코드에 대한 설명을 요청할 수도 있습니다.

* **코드 생성**:
    1.  새 코드 셀을 추가합니다.
    2.  셀 상단에 나타나는 '**생성(Generate)**' 버튼을 클릭합니다. (만약 보이지 않는다면, Colab 설정에서 활성화해야 할 수 있습니다.)
    3.  텍스트 입력창이 나타나면, 만들고 싶은 코드의 기능을 **'자연어(한국어)'**로 설명합니다. 예를 들어, 다음과 같이 입력해 보세요:
``` 1부터 10까지의 숫자 중에서 짝수만 골라서 리스트로 만들고 출력하는 코드 만들어줘
```
    4.  잠시 기다리면 AI가 해당 기능을 수행하는 파이썬 코드를 생성하여 제안합니다.
    5.  제안된 코드를 검토하고, 적절하다면 코드 셀에 삽입하여 실행할 수 있습니다.


* **코드 설명**:
    1.  설명을 원하는 코드가 있는 셀을 클릭하거나, 코드의 특정 부분을 드래그하여 선택합니다.
    2.  셀 상단에 나타나는 '**설명(Explain)**' 버튼을 클릭합니다. (이 버튼은 코드가 셀에 있거나 선택되었을 때 활성화될 수 있습니다.)
    3.  잠시 기다리면 오른쪽 Gemini 채팅창에 해당 코드의 작동 방식이나 의미에 대한 설명이 나타납니다. AI가 생성한 코드나 다른 사람이 작성한 코드를 이해하는 데 도움을 받을 수 있습니다.

### 방법 2: 오른쪽 사이드바 'Gemini' 활용하기

1.  노트북 화면 오른쪽 상단의 **별 모양 아이콘**('**Gemini 열기**')을 클릭합니다.
2.  오른쪽에 Gemini 채팅창이 열립니다.
3.  채팅창에 코드 관련 질문이나 요청 사항을 자연어로 입력합니다. 예를 들어, 다음과 같이 질문해 볼 수 있습니다:

    ```파이썬에서 리스트의 모든 요소를 더하는 방법 알려줘```

    ```이 코드에서 오류가 나는데 왜 그런지 설명해주고 수정해줘: (오류 코드 붙여넣기)```

4.  Gemini가 답변으로 코드 예시, 설명, 수정 제안 등을 제공합니다. 답변 내용을 복사하여 코드 셀에 붙여넣거나 참고하여 코드를 작성/수정할 수 있습니다.

이러한 AI 기능을 활용하면, 파이썬 문법을 잘 모르더라도 원하는 기능을 구현하는 코드를 비교적 쉽게 얻거나, 코드 작성 중 막히는 부분을 해결하는 데 도움을 받을 수 있습니다. 하지만 **AI가 항상 완벽한 코드를 만드는 것은 아니므로, 생성된 코드나 설명을 비판적으로 검토하고 이해하려는 노력이 중요합니다.**
